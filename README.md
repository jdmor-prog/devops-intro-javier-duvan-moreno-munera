# devops-intro-javier-duvan-moreno-munera

DevOps es una forma de trabajar en equipo. Se trata de que los programadores y las personas que se encargan de que el software funcione en los servidores colaboren desde el principio.

El objetivo es que dejen de trabajar por separado y se comuniquen más. Así logran crear y lanzar programas mucho más rápido y con menos problemas.

Una herramienta usada en DevOps es Docker
